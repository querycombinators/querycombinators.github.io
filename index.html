Query Combinators

Query Combinators is a data integration toolkit and methodology designed to let data analysts and other accidental programmers query and analyze complex structured data.

We have published a paper describing the Query Combinators methodology.

  • 2017-Jul-01: the Query Combinators methodology also applies to XML file processing.
  • 2017-Apr-27: an in-memory column-oriented prototype implementation in the Julia Language for Scientific Computing now fully implements the Chicago employee data queries found in our Query Combinators paper
  • 2017-Feb-27: Query Combinators is a paper formally describing this methodology
  • 2015-Dec-18: Query Combinators for TCP:H (src) shows how SQL could be replaced for common analytic business queries
  • 2015-Dec-10: Query Combinators over Employee Data (src) shows interesting yet succinct analysis on City of Chicago's employee data
  • 2008-2013: this methodology was proven to work "in practice" with our HTSQL url2sql translator.
